MovieMadness

Welcome!

The following project web scrapes the IMDB website for movies in theaters this week, using the Beautiful Soup python library. A similar email is auto-generated weekly to notify you on movies in theaters this week.

For more information, visit the IMDB website at:
https://www.imdb.com/movies-in-theaters/
